## 概要

- L05 では「金融政策の操作変数は金利（コールレート）であり、イールドカーブを通じて長期金利・実体経済へ波及する」という**伝達メカニズム**を扱った。本講はその前提に立ち、「その名目短期金利をどのようなルールで動かすべきか」を扱う。
- 前半：FRBの政策行動を近似した**テイラー・ルール**を軸に、金融政策ルールの意義・定式化・有用性・留意点を整理。
- 後半：中央銀行の**損失関数**を定義し、NKPCを制約条件として損失を最小化する**最適金融政策**を理論的に導出。テイラー・ルールとの対比で「規範的なルールとは何か」を掘り下げる。
- 全講を通じた位置づけ：「ルールvs裁量」の議論（L03で登場）と「ニューケインジアン理論」（L04）が交差する講。

## レジュメ要点

### 1. 金融政策ルールの背景：「ルール vs 裁量」の議論

- **フリードマンのk%ルール**：マネタリズムの代表として、マネー供給量を一定率 k% で伸ばす機械的ルールを提唱。背景は自然失業率仮説＋合理的期待（L03）。「裁量的政策は有効でなく機械的ルールの方が望ましい」という理論状況から生まれた。
- ただし現実の中央銀行は操作変数がマネーではなく**金利**だったため、このルールを採用した中央銀行は皆無。
- 1993年、スタンフォード大学の**ジョン・テイラー教授**が「FRBの名目短期金利（FFレート）は、インフレ率と実質GDPのトレンドからの乖離に一定のルールで反応しているように見える」と指摘 → **テイラー・ルール**と呼ばれる。

### 2. テイラー・ルールの原型（1993年論文）

テイラーが示した原型式（スライド記載）は、名目短期金利 $R$ を次の4項の和で近似：

$$R = P + 0.5Y + 0.5(P - 2) + 2$$

| 記号 | 意味 |
|---|---|
| $R$ | 名目短期金利（FFレート） |
| $P$ | インフレ率 |
| $Y$ | 実質GDPのトレンドからの乖離率 |
| $2$（固定項） | テイラーが想定したFRBの暗黙の均衡金利（自然利子率の代理） |
| $2$（目標） | テイラーが推測したFRBの目標インフレ率 |

- このシンプルな式でも、1987〜92年頃の実際のFFレートの動きをかなりよく近似できた。

### 3. テイラー・ルールの一般的な定式化

$$i_t = \pi^* + r_t^n + \phi_\pi(\pi_t - \pi^*) + \phi_y y_t$$

| 記号 | 意味 |
|---|---|
| $i_t$ | 名目短期金利（日本：コールレート、米国：FFレート） |
| $\pi^*$ | 中央銀行の目標インフレ率 |
| $r_t^n$ | 自然利子率（時変。経済成長力で決まる実質均衡金利） |
| $\pi_t$ | インフレ率 |
| $y_t$ | 需給ギャップ |
| $\phi_\pi$ | インフレ率への反応度 |
| $\phi_y$ | 需給ギャップへの反応度 |

- $\pi^* + r_t^n$：名目金利の**長期均衡値**（フィッシャー方程式）。
- $\phi_\pi(\pi_t - \pi^*)$：インフレ率が目標値を超えれば金利を引き上げ（金融引き締め）、下回れば引き下げ（金融緩和）。
- $\phi_y y_t$：景気（需給ギャップ）がプラスなら引き締め、マイナスなら緩和。
- テイラー1993年の原型：$\pi^* = 2\%$、$r_t^n = 2\%$（一定として扱う）、$\phi_\pi = 1.5$、$\phi_y = 0.5$。
  - スライドでは $r_t^n = 4\%$（＝目標インフレ率2%＋実質均衡金利2%の合計）と整理されている場合もある。
- 自然利子率 $r_t^n$ は長期的に一定と考えることもあるが、短期的に需要変動等で変化するため、実践では時変として扱うことが多い。

### 4. テイラー・ルールの意義

- テイラーはあくまで「FRBの事後的な行動パターンの近似」として提示し、機械的なルール運営を主張したわけではない。
- 中央銀行から高い関心を持たれた理由：
  1. **操作変数が金利**：実務と一致（k%ルールはマネーが前提で現場と乖離していた）。
  2. 「インフレ上昇→利上げ、インフレ低下→利下げ」という**典型的な政策パターン**を数式化。
  3. 物価安定と同時に**景気変動（需給ギャップ）にも対応**するパターンを表現。
- テイラーのスタンス：ベンチマークとして持つことで「どのような政策行動がマクロ経済安定に寄与するか」を検討しやすくなる。

### 5. テイラー・ルールが発展した学問的背景

- 1990年代後半以降、テイラー・ルールをベンチマークとした金融政策ルール研究が**学界と中央銀行の双方で急増**。
- 背景：**ニューケインジアン理論の台頭**（L04）。
  - ルーカス・フリードマン流の理論では「ルールに基づく政策は全て人々に予見され実体経済に無影響」→ どのルールを選んでも結果は同じ→ ルール研究は無意味。
  - ニューケインジアン理論では「**価格の粘着性**があるため、予見されたルールであっても実体経済への影響力を持つ」→ 適切なルール選択が経済・物価の安定性を高められる→ ルール研究に意義。
- テイラー教授自身は1970年代末から名目賃金の硬直性に基づくモデルを研究していたが、当時の学界環境（フリードマン・ルーカス優勢）では評価されにくかった。標準的ニューケインジアン理論の確立（1990年代末〜）でテイラー・ルール研究も大きく発展。

### 6. 日本におけるテイラー・ルールの当てはまり

- 日本のデータ（CPIインフレ率・日銀の需給ギャップ・潜在成長率を自然利子率の代理変数）でテイラー・ルールを計測：
  - **1985〜1990年代後半まで**：コールレートとテイラー・ルールは大まかに似た推移。
  - **1990年代末以降**：**ゼロ金利制約**により乖離が増大（テイラー・ルール上は大幅マイナス金利が示唆されても、現実のコールレートはゼロ近傍に留まる）。
- 計測に用いたパラメータ：目標インフレ率は2012年末まで1%、2013年初以降2%（日銀が2%目標を設定したタイミング）。

### 7. テイラー・プリンシプル

$$\phi_\pi > 1$$

- テイラー・ルールにおいてインフレ率への反応度が**1を超える**こと（$\phi_\pi > 1$）。

- **なぜ重要か**：インフレ率が目標値から1%上昇した時、名目金利を**1%超**引き上げる。
  - このとき実質金利（＝名目金利－インフレ率）が**上昇**→ 実体経済に金融引き締め効果。
  - 逆に $\phi_\pi < 1$ だと、インフレ率上昇に対して実質金利が低下してしまい、インフレがさらに悪化しうる（不安定）。
- 多くの経済モデルで、インフレ率を長期的に中央銀行の目標値に**収束させるための必要十分条件**として知られている。
- 最低限これを満たすことが規範的に要求される。

### 8. 金利スムージングを導入したテイラー・ルール

$$i_t = \rho\, i_{t-1} + (1-\rho)\left[\pi^* + r_t^n + \phi_\pi(\pi_t - \pi^*) + \phi_y y_t\right]$$

| 記号 | 意味 |
|---|---|
| $i_{t-1}$ | 前期の名目短期金利 |
| $\rho$ | 金利スムージングのパラメータ（0〜1。大きいほど金利変化がなだらか） |

- 現実の中央銀行は急激な金利変更を避ける傾向（**漸進的な変更**）。理由：
  1. 急激な変更が金融市場を混乱させるリスクを回避。
  2. 一方向の変更（例：引き締め）を段階的に行うと示すことで、市場の先行き予想が安定し**政策効果を強める**。
- $\rho$ と $(1-\rho)$ の加重平均：前期金利水準（ウェイト$\rho$）と通常テイラー・ルールが示す水準（ウェイト$(1-\rho)$）の間を取る。
- 日本のデータへの当てはめでは、$\rho = 0.6$ と設定することでスムージングなしよりコールレートへの当てはまりが**大きく向上**。

### 9. 金融政策ルールの有用性

#### ① 予測可能性（predictability）

- L05で示した通り、金融政策の効果は短期金利→（イールドカーブ経由）→長期金利の波及が起点。
- 市場参加者が「将来の短期金利経路」を中央銀行の政策意図に沿って予測できれば、長期金利が安定し政策効果が高まる。
- テイラー・ルールのようにシステマティックな政策運営が行われているという認識が形成されると、将来の政策への**予測可能性が高まる**。
- さらにテイラー・プリンシプルが満たされることが期待に織り込まれれば、「インフレ上昇→中央銀行が引き締め→インフレ収束」という予想から、**期待インフレ率が安定**する効果もある。

#### ② 政策のベンチマーク

- 中央銀行自身にとって、インフレ率・需給ギャップのデータを当てはめて「テイラー・ルール上の水準」を参照値として確認できる。
- ルールからの乖離が必要な局面（例：資産価格の急変、市場ボラティリティ上昇）で、「なぜ・どの程度乖離しているか」を整理 → **市場との対話（コミュニケーション）ツール**として機能。

### 10. 金融政策ルールの留意点

1. **リアルタイム・データ問題**
   - 自然利子率・需給ギャップは直接観測不可。推計手法次第で値が変わる。
   - GDP統計などが改定されると、事後的に自然利子率・需給ギャップの値も変化（リアルタイムに把握していた値が事後的に変わる）。
   - インフレ率のような通常観察されるデータでも基準改定により事後的変化が起こりうる。

2. **テイラー・ルールは必ずしも「最適」ではない**
   - あくまで過去の平均的な政策パターンの近似。$\phi_\pi = 1.5$、$\phi_y = 0.5$ が最適係数とは限らない。
   - インフレ率・需給ギャップ以外の変数に反応する方が良いケースもある。
   - 望ましいルールを求めるには、マクロ経済モデルのもとで各種ショック（需要・供給ショック）を想定した上でマクロ変動を最も安定化させるルールを探索する必要がある。

### 11. 中央銀行によるマクロ経済の安定化

- 中央銀行の目標：
  1. **物価の安定**（インフレ率を目標値に近づける）。
  2. **需給ギャップの安定化**（供給ショック等による景気変動を緩和）。
     - 供給ショックが起きると「インフレ安定だけを追求すると景気変動が大きくなる」というトレードオフが生じる（L02）。
- 金融政策でできること・できないこと：
  - **できないこと**：GDPの長期トレンド（潜在GDP・自然産出量）を変えること。これは技術進歩・人口成長等の供給側要因で決まる。
  - **できること**：価格の粘着性等により生じる「GDPのトレンドからの短期的乖離（需給ギャップ）」に影響を与えること。

### 12. 中央銀行の損失関数

$$L_t = (\pi_t - \pi^*)^2 + \alpha\, y_t^2$$

| 記号 | 意味 |
|---|---|
| $L_t$ | 中央銀行の $t$ 期における損失 |
| $\pi_t - \pi^*$ | インフレ率の目標値からの乖離 |
| $y_t$ | 需給ギャップ |
| $\alpha$ | 需給ギャップの安定を重視する度合い（ウェイト） |

- 2乗にする理由：①目標値からの乖離はプラス・マイナスどちらも等しく不望ましいため（両方をペナルティ化）、②乖離幅が大きいほど損失が急増（ハイパーインフレのような極端な乖離は特に大きなコスト）。
- 中央銀行はこの損失をできるだけ小さくするように政策運営すると想定。

#### 中央銀行の損失関数（無差別曲線）

- 縦軸：インフレ率 $\pi_t$、横軸：需給ギャップ $y_t$。
- 2次の損失関数では、**等損失の点の集合**（無差別曲線）は中心 $(y_t, \pi_t) = (0, \pi^*)$ の**楕円形**。
- 内側の楕円ほど損失が小さい。中央銀行は取り得る $(\pi_t, y_t)$ の組み合わせの中で、最も内側（損失最小）の楕円に接する点を実現するのが最適。

### 13. インフレ率と需給ギャップのトレードオフ

ニューケインジアン・フィリップス曲線（NKPC）：

$$\pi_t = \beta E_t \pi_{t+1} + \kappa y_t + \mu_t$$

| 記号 | 意味 |
|---|---|
| $\beta$, $\kappa$ | 正のパラメータ |
| $\mu_t$ | コストプッシュ・ショック（原油価格変動等） |

- コストプッシュ・ショック $\mu_t > 0$ のとき、同じ需給ギャップ水準でもインフレ率が押し上げられる。
- 単純化のため $E_t \pi_{t+1} = \pi^*$（将来インフレ期待は目標値に一致）と仮定。$\mu_t = \mu > 0$（一定の正値）のとき：

$$\pi_t = \kappa y_t + \underbrace{\beta \pi^* + \mu}_{\text{正の一定値}}$$

- これはグラフ上で**右上がりの直線**（傾き$\kappa$）で表される。
- コストプッシュ・ショックの値次第でNKPCが上下に**平行移動**し、インフレ率と需給ギャップの間にトレードオフが生じる。
- 中央銀行はこのNKPC上にある点（取り得る組み合わせ）の中で、損失関数の楕円が接する点（損失最小点）を選ぶのが最適政策。

### 14. 最適金融政策の導出（ターゲット条件）

損失関数の最小化問題：

$$\min_{\pi_t, y_t} \left[(\pi_t - \pi^*)^2 + \alpha y_t^2\right] \quad \text{s.t.} \quad \pi_t = \kappa y_t + \beta \pi^* + \mu$$

ラグランジュ乗数法による解法：

① ラグランジアンを定義：

$$\mathcal{L}_t = (\pi_t - \pi^*)^2 + \alpha y_t^2 - \lambda_t\left[\pi_t - \kappa y_t - \beta \pi^* - \mu\right]$$

② $\pi_t$ と $y_t$ で偏微分してゼロに置く：

$$\frac{\partial \mathcal{L}_t}{\partial \pi_t} = 2(\pi_t - \pi^*) - \lambda_t = 0$$

$$\frac{\partial \mathcal{L}_t}{\partial y_t} = 2\alpha y_t + \lambda_t \kappa = 0$$

③ $\lambda_t$ を消去すると**最適金融政策の条件式（ターゲット条件）**：

$$\boxed{\alpha y_t = -\kappa(\pi_t - \pi^*)}$$

- グラフ上では中心 $(0, \pi^*)$ を通る**右下がりの直線**（傾き $-\alpha/\kappa$）。
- コストプッシュ・ショックがどう変化してNKPCが上下に移動しても、この緑線（ターゲット条件）との交点を実現するように政策運営すれば損失関数を最小化できる。

### 15. 最適性条件を満たす金融政策ルールの導出

動学的IS曲線：

$$y_t = E_t y_{t+1} - \sigma(i_t - E_t \pi_{t+1} - r_t^n)$$

金利について整理：

$$i_t = E_t \pi_{t+1} + r_t^n + \frac{1}{\sigma}(E_t y_{t+1} - y_t)$$

ここにターゲット条件 $\alpha y_t = -\kappa(\pi_t - \pi^*)$ を代入すると：

$$i_t = E_t \pi_{t+1} + r_t^n + \frac{1}{\sigma} E_t y_{t+1} + \frac{\alpha}{\sigma \kappa}(\pi_t - \pi^*)$$

- テイラー・ルールと**似た形状**だが、係数はモデルの構造パラメータ（IS曲線の $\sigma$、NKPCの $\kappa$、損失関数のウェイト $\alpha$）に依存して内生的に決まる。
- テイラー・ルールの係数 $\phi_\pi = 1.5$、$\phi_y = 0.5$ などは最適性から導かれた値ではなく、あくまで過去のパターンの近似。

### 16. 最適金融政策を追求することのメリット

1. **規範的な思考体系**：評価基準（損失関数）を明確化し、経済構造の知識に基づいて最良の状態を実現しようとすること自体が、金融政策運営として最も規範的な経済学的アプローチ。家計・企業の効用・利潤最大化と同じ発想。
2. **調査研究の促進**：最適政策の実現にはフィリップス曲線の傾き $\kappa$、IS曲線のパラメータ $\sigma$ 等の経済構造を正確に把握する必要 → 調査研究活動が促進され、政策の質が向上。テイラー・ルールでは必ずしも不要なそうした知識を、最適政策の視点は要求する。

### 17. 最適金融政策を追求することの留意点

1. **損失関数の恣意性**：ウェイト $\alpha$ をどう設定するかは裁量的。インフレと需給ギャップ以外の変数（金融市場指標等）を損失関数に含めるかも論点。
2. **経済構造の不確実性**：正確な経済構造の知識（NKPCの $\kappa$、IS曲線の $\sigma$ 等）を常に持つ前提は現実には満たされない。マクロ経済は複雑で単純モデルでは描写しきれず、構造自体も時変の可能性。最大限の調査研究でも不確実性は残る → **不確実性のもとでの最適政策**を別途考える必要（第7講へ続く）。

## 授業での補足

- **テイラー教授の人物像と実務感覚**：テイラーは純粋な学者ではなく中央銀行の実務にも精通した研究者。だからこそ「操作変数はマネーではなく金利」という実態を出発点にしたルール研究が現実の中央銀行に受け入れられた。
- **「暗黒の時代」という表現**：テイラー教授自身が1970〜80年代の金融政策研究の低迷期を「暗黒の時代」と呼んでいた。ルーカス・フリードマン理論が支配的だった時代、金融政策の有効性そのものが否定されていた。
- **テイラー・ルールと日本のバブル**：日本のテイラー・ルール計測のグラフに関連して、1985〜90年頃の実際のコールレートの動きが大まかにテイラー・ルールと近似していたとも読めるが、これはバブル生成・崩壊期を含む。「大まかに見ると近似していた」という慎重な表現。
- **ゼロ金利制約の明示**：1990年代末以降の乖離はゼロ金利制約に起因。テイラー・ルールが示唆するマイナス金利の大幅な水準に政策金利は下げられなかった。
- **「スムージングのパラメータ $\rho = 0.6$」の注釈**：日本のデータへの当てはめで設定した値。これにより当てはまりが大きく改善 → 現実の中央銀行が漸進的な金利操作を行っていることの傍証。
- **テイラー・プリンシプルと実質金利の関係の丁寧な説明**：「インフレ率が1%上昇→名目金利を1%超引き上げ→実質金利が上昇→引き締め効果」という論理チェーンを武藤氏が講義で丁寧に確認。テイラー・プリンシプルが満たされないと「インフレ上昇→実質金利低下→さらにインフレ悪化」というスパイラルに。
- **損失関数のウェイト $\alpha$ の解釈**：$\alpha$ を大きくするほど「需給ギャップの安定（景気安定）」を相対的に重視。ただしその分、短期的にインフレ率の安定を犠牲にすることになる。
- **最適金融政策の数式は「覚える必要なし」との明示**：ラグランジュ乗数法による導出式やターゲット条件を満たすルールの式は、武藤氏が「覚える必要はない、パラメータへの依存を理解すればよい」と明言。ただし**ターゲット条件** $\alpha y_t = -\kappa(\pi_t - \pi^*)$ の意味は理解必須。
- **武藤氏の個人的な関心**：「金融政策ルールと最適金融政策は金融政策研究の一番熱い部分で、自分が非常に関心が高い部分」と言及。現在も新しい研究が続く活発な分野と強調。
- **インフレ率の2乗の意味の補足**：スライドには「上方・下方乖離のどちらも不望ましい」と書かれているが、講義では「ハイパーインフレは乖離幅が非常に大きいため損失が急増するという性質も2乗で表現できる」という補足説明あり。

## 理解したこと

1. **テイラー・ルールの本質は「FRBの行動記述」から「政策ベンチマーク」へ**：テイラー自身は機械的なルール運営を主張せず、あくまで事後的な記述（positive）として提示した。ただしその「名目短期金利＋インフレ反応＋需給ギャップ反応」という構造が現実に即していたため、規範的な（normative）政策ベンチマークとしても発展した。

2. **テイラー・プリンシプル（$\phi_\pi > 1$）は最低限の安全弁**：インフレ上昇に対して実質金利が上昇する（＝引き締め）ためには、名目金利の反応度が1を超えなければならない。これが満たされないとインフレが自己実現的に加速しうる。

3. **損失関数＋NKPCでトレードオフを可視化**：供給ショック（コストプッシュ）があるとNKPCが上方シフトし、インフレ安定と景気安定の両立が難しくなる。楕円の無差別曲線とNKPCの接点がそのトレードオフのもとでの「最善点」。ターゲット条件はそうした接点の軌跡を表す。

4. **最適政策ルールの係数はテイラー係数と同じではない**：最適性から導かれる名目金利ルールの係数は IS曲線の $\sigma$、NKPCの $\kappa$、損失ウェイト $\alpha$ に依存する。テイラー・ルールの慣例的な係数（1.5・0.5）はこれらのパラメータを正確に反映したものではなく、最適性を保証しない。

---

## 未理解点

- **損失関数のウェイト $\alpha$ の実践的な決め方**：どの中央銀行も $\alpha$ を明示していない。インフレーション・ターゲティングは「物価安定が第一義」のように見えるが、実際の政策運営では需給ギャップにもかなり目配りしている。両者の相対的なウェイトは何らかの方法で推定されているのか、あるいは完全に裁量的なのか。
- **「将来インフレ期待は目標値に一致」という仮定の現実適合性**：NKPC分析の単純化仮定として採用したが、実際の期待形成はアダプティブ（適応的）な要素も持つ。期待のより精緻な扱いで最適政策のターゲット条件はどう変わるか。
- **第7講との接続**：武藤氏が「不確実性がある下での最適金融政策」を次回で扱うと宣言。経済構造の不確実性（$\kappa$・$\sigma$ の推計誤差）はロバストネスの問題を引き起こすが、具体的にどのような修正を迫るのか。
