# L04 ニューケインジアン理論と金融政策

**授業**: 金融政策1
**日付**: 2026-04-28

---

## 概要

- フリードマン/ルーカスの後を受けて1990年代末〜2000年代に確立した**ニューケインジアン理論**を導入する回
- 「価格の粘着性」を中核に置き、金融政策が実体経済に持続的影響を持つメカニズムを定式化する
- 標準的なニューケインジアン・モデル（金融政策＋動学的IS曲線＋NKPC）の3本式まで到達し、金融引き締めショックへの反応を概観する
- 現代の先進各国中央銀行の政策運営の理論的基盤がここにある

---

## レジュメ要点

### ニューケインジアン理論の台頭

- **ニューケインジアン理論**は「価格の粘着性（sticky price）」の概念を中心に据えるマクロ経済学の理論
  - 金融政策やインフレーションを分析する**現代の標準的な理論的枠組み**
- 価格の粘着性自体は1980年代半ばにマクロ経済学に導入されたが、現在の標準的な金融政策分析の枠組みとして確立したのは**1990年代末〜2000年代**
- 中心的貢献者:
  - **マイケル・ウッドフォード**（コロンビア大学）
  - **ジョルディ・ガリ**（ポンペウ・ファブラ大学／スペイン）
  - **ローレンス・クリスチアーノ**（ノースウェスタン大学）
- 多数の学者が貢献しており、扱う研究者が広範に渡るのが発展の特徴
- 現代の先進各国中央銀行の金融政策運営の理論的基盤は、ニューケインジアン理論に基づくと言って過言ではない

### 金融政策理論の比較：フリードマン/ルーカス vs ニューケインジアン

| | フリードマン/ルーカス | ニューケインジアン |
|---|---|---|
| 実体経済への影響源泉 | **予期されないインフレ** | **価格の粘着性** |
| 価格調整 | 瞬時的に行われる | 時間がかかる（粘着的） |
| 金融政策の実体経済への影響 | ごく一瞬のみ | 価格調整の間は継続的に影響 |
| 政策の役割 | 安定化の役割なし、むしろ攪乱要因 | 経済・物価を安定させる役割を持つ |
| 期待の方向 | バックワードルッキング | **フォワードルッキング** |
| 将来政策のアナウンス | 効果なし | 現在の経済・物価に影響を及ぼせる（効果的） |

- ニューケインジアン理論はケインズ経済学を受け継ぐ形で価格の粘着性を導入している点が特徴
- 「将来の金融政策運営も現在の経済・物価に影響を及ぼせる」というルートが新たに開かれた点が重要

---

### 価格の粘着性

- **定義**: 企業が価格設定を行う際、必ずしも毎日自由に価格を変更できず、何らかの理由から価格改定を時々（または一定の決まったタイミングで）しか行えないという仮定
- **粘着性の理由**:
  - **メニュー・コスト**（値札更新に伴うコスト）
  - 頻繁な価格変更による顧客離れ
  - その他様々な要因

#### 1か月間で価格が更新される商品の割合

| | 商品全体 | モノ | サービス |
|---|---:|---:|---:|
| 日本 | 25% | 23% | 4% |
| 米国 | 25% | 22% | 15% |
| ドイツ | 14% | 5% | 4% |
| フランス | 21% | 18% | 7% |
| イタリア | 10% | 6% | 5% |

（注）欧州中央銀行が2002年に組織した Inflation Persistence Network（IPN）による調査結果。
（出所）渡辺努（2022）『物価とは何か』、講談社、p.243

- 1か月間で価格が変わる商品は商品全体の約1/4にとどまり、3/4は据え置かれている
- 特に**サービスはモノより粘着的**（日本では4%、米国でも15%）
- この傾向は日本に限らず先進各国共通 → 価格の粘着性という仮定は現実的

### 2つの世界：「現実の経済」と「長期均衡」

ニューケインジアン理論を理解するには2つの世界を区別する必要がある。

| | 現実の経済 | 長期均衡 |
|---|---|---|
| 性質 | 価格の粘着性がある | 価格が完全に伸縮的（仮想的） |
| 観察可能性 | 現実のデータで観察できる | 直接観察できない（潜在的水準） |

この2つの世界の間には**「ギャップ」**が存在する。

#### 主要な変数は3つ：実質金利、産出量、インフレ率

| 変数 | 現実の経済 | 長期均衡（潜在水準） | ギャップ |
|---|---|---|---|
| 実質金利 | 実質金利 | **自然利子率**（均衡実質金利） | 実質金利ギャップ |
| 産出量 (GDP) | 産出量 | **自然産出量**（潜在GDP） | 需給ギャップ |
| インフレ率 | インフレ率 | 中央銀行の目標インフレ率 | 目標からの乖離 |

- 長期均衡における実質金利・産出量は**金融政策の直接的影響を受けない**潜在水準
- 自然利子率・自然産出量は直接観察できないため、ニューケインジアン・モデルに基づく**推計**が必要
  - 中央銀行のエコノミストが分析・推計を行っている

> **グラフ: 米国の自然利子率の推計値（ウィリアムズ等）**
> 縦軸=自然利子率（％、0〜6）、横軸=年（1960〜2020）。過去も変動を続けてきたが、近年にかけて低下トレンド（下方シフト）。

> **グラフ: 米国の需給ギャップの推計値**
> 縦軸=需給ギャップ（％、−8〜+8）、横軸=年（1960〜2020）。景気後退期に大きくマイナス、回復期にプラスへ振れる循環を示す。

### 標準的なニューケインジアン・モデルの3つの構成要素

1. **金融政策**：実質金利を決定
2. **IS曲線**：需給ギャップを決定
3. **ニューケインジアン・フィリップス曲線（NKPC）**：インフレ率を決定

---

#### ① 金融政策：名目金利を通じて実質金利をコントロール

- 中央銀行は**名目金利**を操作することにより、**実質金利**に影響を与えて金融政策を実施する

##### フィッシャー方程式（実質金利と名目金利の関係式）

$$\text{実質金利} = \text{名目金利} - \text{期待インフレ率}$$

- 価格が粘着的なもとでは期待インフレ率は短期的に大きく変化しない
- そのため中央銀行は名目金利を操作することで、実質金利の水準をある程度自分の意図するように決められる

---

#### ② IS曲線：実質金利ギャップにより需給ギャップを決定

$$\text{需給ギャップ} = -\alpha (\text{実質金利} - \text{自然利子率})$$

| 記号 | 意味 |
|---|---|
| 需給ギャップ | 実際の産出量 − 自然産出量 |
| 実質金利 − 自然利子率 | 実質金利ギャップ |
| α | 正のパラメータ |

- 中央銀行が実質金利を**低く**すると → 借入コスト低下 → 経済活動活発化 → **景気は過熱気味（金融緩和）**
- 中央銀行が実質金利を**高く**すると → 借入コスト上昇 → 経済活動鈍化 → **景気は悪化気味（金融引き締め）**
- **自然利子率**は景気中立的な実質金利の水準
  - 実質金利＝自然利子率のとき需給ギャップ＝0
  - 金融政策以外の要因（**技術進歩率・人口増加率**などの経済のファンダメンタルズ）で決まる
  - 中央銀行が決められない外生変数

---

#### ③ ニューケインジアン・フィリップス曲線（NKPC）：インフレ率を決定

$$\pi_t = \beta \, E_t \pi_{t+1} + \kappa \, y_t$$

| 記号 | 意味 |
|---|---|
| $\pi_t$ | 当期のインフレ率 |
| $E_t \pi_{t+1}$ | 来期のインフレ率に対する**当期に形成するフォワードルッキングな期待**（合理的期待） |
| $y_t$ | 需給ギャップ |
| $\beta$ | 将来割引因子（$\beta < 1$） |
| $\kappa$ | 正のパラメータ |

- インフレ率は **(i) 将来のインフレ率に対するフォワードルッキング期待** と **(ii) 需給ギャップ** で決まる
- 需給ギャップ > 0（景気過熱）→ インフレ圧力／< 0（景気悪化）→ デフレ圧力

##### フリードマン/ルーカスのフィリップス曲線との比較

$$\pi_t = E_{t-1} \pi_t + \kappa \, y_t$$

- 期待は**バックワードルッキング**（過去に形成した当期インフレ率の予想）
- 移項すると当期のインフレ率に関する予測誤差 $\pi_t - E_{t-1}\pi_t$ が需給ギャップを決める形になる
- → **予期されないインフレ**だけが実体経済に影響を与えるという主張に対応

##### 数式的にはわずかな違いだが本質的に重要

- NKPCは「**将来から現在へ**」、フリードマン/ルーカスは「**過去から現在へ**」期待が向く
- この違いから、将来の金融政策アナウンスが現在に影響するかどうかが分かれる

---

### NKPCで期待がフォワードルッキングになる理由

**問**: なぜNKPCでは期待がフォワードルッキングになるのか？

**答**: 企業は粘着性により一定頻度（例：1年に1回）でしか価格改定できない。改定時には**現在の景気情勢だけでなく、先行き価格を据え置いている期間の景気情勢をフォワードルッキングに予想**したうえで、適切な価格を設定することが合理的になるから。

| 状況 | 価格決定で参照する景気情勢 |
|---|---|
| 価格を毎期自由に決められる場合 | 現在の景気情勢のみ |
| 価格を一定頻度でしか変更できない場合 | 現在 + **先行き価格据置期間の景気情勢** |

#### NKPCの逐次代入による展開

NKPCを将来へ進めて期待値を取り、繰り返し代入すると:

$$\pi_t = \kappa \left( y_t + \beta E_t y_{t+1} + \beta^2 E_t y_{t+2} + \beta^3 E_t y_{t+3} + \cdots \right)$$

- 現在のインフレ率は、**現在から将来にわたる景気情勢（需給ギャップ）へのフォワードルッキング期待**に依存
- 価格の粘着性 → フォワードルッキング期待が必要 → NKPCの形

---

### ニューケインジアン・モデルにおける金融政策の効果

中央銀行が現在のインフレ率に影響を与えるルートは2つある。

#### ルート1：現在の名目金利を通じる経路

```
現在の名目金利 → 現在の実質金利ギャップ → (IS曲線) → 現在の需給ギャップ → (NKPC) → 現在のインフレ率
```

例: 名目金利を引き下げる → 実質金利ギャップ低下 → 需給ギャップ上昇 → インフレ率上昇

#### ルート2：将来の金融政策への期待を通じる経路

```
将来の名目金利への期待 → 将来の実質金利ギャップへの期待
  → 将来の需給ギャップへの期待 → 将来のインフレ率への期待
  → (NKPCのフォワードルッキング項) → 現在のインフレ率
```

例: 「将来は緩和的になる」との期待形成 → 将来の景気・インフレ期待上昇 → 現在のインフレに上昇圧力

→ **将来の政策運営のアナウンスメント**が現在の経済・物価に効くという、フリードマン/ルーカスにはない含意

### 標準的ニューケインジアン・モデルの3本式

#### 金融政策（テイラー・ルール）

$$i_t = \pi^* + r_t^n + \phi_\pi (\pi_t - \pi^*) + \phi_y \, y_t + \nu_t$$

- ジョン・テイラーが定式化。現実の中央銀行の政策パターンに近い
- 中央銀行は **(i) 目標インフレ率からの乖離**、**(ii) 需給ギャップ** に応じて名目金利を反応させる
- 第6回講義で詳述予定

#### 動学的IS曲線

$$y_t = E_t y_{t+1} - \sigma (i_t - E_t \pi_{t+1} - r_t^n)$$

- 需給ギャップは**将来の需給ギャップへの期待** $E_t y_{t+1}$ にも依存（この点が「動学的」）
- 実質金利ギャップ部分は静的IS曲線と同じ本質
- **消費者の最適化行動**（予算制約のもとでの効用最大化）から導出される

#### NKPC（コストプッシュ・ショック付き）

$$\pi_t = \beta \, E_t \pi_{t+1} + \kappa \, y_t + \mu_t$$

- $\mu_t$: **コストプッシュ・ショック**
  - 需給ギャップ以外でインフレ率を変動させる要因
  - 例: 原油価格変動（中東情勢）、食料原材料価格
  - これがあると**インフレ率の安定と需給ギャップの安定にトレードオフ**が生じる
- **企業の最適化行動**（粘着価格下での価格設定）から導出される

#### 記号一覧

| 記号 | 意味 |
|---|---|
| $i_t$ | 名目金利 |
| $y_t$ | 需給ギャップ |
| $\pi_t$ | インフレ率 |
| $r_t^n$ | 自然利子率 |
| $\pi^*$ | 中央銀行の目標インフレ率 |
| $\nu_t$ | 金融政策ショック |
| $\mu_t$ | コストプッシュ・ショック |
| $\phi_\pi, \phi_y, \sigma, \beta, \kappa$ | 正のパラメータ |

#### ルーカス批判との関係

- 動学的IS曲線とNKPCは**消費者・企業のミクロ的最適化行動から導出**されている
- 経験則的な関係式に依拠していないため、**ルーカス批判を免れている**
- これがニューケインジアン・モデルが各国中央銀行で広く使われるようになった理由のひとつ

---

### 金融政策ショックへの反応（金融引き締めの例）

> **グラフ: 金融引き締めショックへのインパルス応答（Galí 2008より）**
> 4つのパネル（名目金利／実質金利／需給ギャップ／インフレ率）を時間軸でプロット。すべて定常値からの乖離。
> - 0期に名目金利が引き上げられ、その後数期かけて徐々に縮小
> - 実質金利も同方向に上昇（実質金利ギャップ＞0）
> - 需給ギャップは負方向に低下（景気悪化）
> - インフレ率は低下

#### 因果連鎖

```
名目金利↑ → 実質金利ギャップ↑ → (動学IS) → 需給ギャップ↓ → (NKPC) → インフレ率↓
```

- 中央銀行が現実に想定する金融引き締めの効果と整合的
- ルーカス批判をクリアした理論モデルが、中央銀行員の頭の中の経済像をきちんと表現できている

## 授業での補足

- **「無風」のような重要な強調**:
  - 「フォワードルッキングな期待が現在のインフレ率に影響を与える点は、強調しすぎてもしすぎることはないぐらい重要」
  - 一見フリードマン/ルーカスの式と似ているが、期待のタイミングが反対方向を向いているという「微妙な違いが実は非常に重要」
- **ケインズ経済学との連続性**:
  - 古典派は需給調整時に価格が完全に動くと考える
  - ケインズは「価格は短期的にすぐには動かない」とした
  - ニューケインジアンはこのケインズ的発想を**価格の粘着性**として明示的に導入し、最適化行動から導出した
- **値段の体感例**（粘着性の現実感を補強）:
  - 床屋の値段、クリーニングの値段は「なかなかその頻度では変わらない」
  - 一方で生鮮食品やガソリンスタンドのオイル価格は日々変わる → ただしそういう商品の割合は全体では多くない
- **自然利子率の推計者**: ニューヨーク連銀総裁の**ジョン・C・ウィリアムズ**（元エコノミスト）が作ったモデル
  - シミュレーションで「価格の粘着性が0だったら過去の実質金利はどう動いていたか」を推定
- **目標インフレ率の例**: 多くの中央銀行で**2%**
  - インフレ率が目標値より上ぶれると、$\phi_\pi$ を係数として名目金利を引き上げ → 引き締めへ
- **動学的IS曲線の「本質は静的版と同じ」**: $E_t y_{t+1}$ が入ってる分だけ動学化しているが、実質金利ギャップが需給ギャップを動かす部分は変わらない
- **コストプッシュ・ショックの直感**: 国内景気と無関係にインフレ率を動かす要因（中東情勢起因の原油価格など）。**インフレ厳格安定 vs 需給ギャップ安定** のトレードオフを生む
- **次回（第5講）への接続**: 金融政策の効果波及経路。テイラー・ルールの詳細は第6講

## 理解したこと

- フリードマン/ルーカスは「価格は瞬時に動く」「予期されないインフレだけが効く」と仮定したため、金融政策は短期的・偶発的にしか実体経済に影響できなかった
- ニューケインジアンは「価格は粘着的」「期待はフォワードルッキング」とすることで、(i) 名目金利→実質金利→需給ギャップの直接ルートと、(ii) 将来政策期待→将来インフレ期待→現在インフレの期待ルート、の2系統を金融政策に与えた
- 「2つの世界」（現実 vs 長期均衡）の発想は、現実に観察できる変数と「もし価格が完全に伸縮的だったら」の潜在水準を分けて考えることで、政策の効きどころを明確にする概念装置
- NKPCのフォワードルッキング期待は、企業が「価格を据え置く期間の景気」を予想して合理的に価格設定する行動から自然に出てくる
- 動学的IS曲線とNKPCがミクロ最適化から導かれていることが**ルーカス批判への防御**になっており、これが各国中央銀行に採用された決定的理由

---

## 未理解点

- 動学的IS曲線の導出（消費者の効用最大化からのオイラー方程式の展開）の数学的詳細
- NKPCの「カルボ価格設定」など具体的な粘着性のミクロ的定式化（講義では明示されなかった）
- テイラー・ルールの $\phi_\pi, \phi_y$ の典型的な数値とテイラー原理（$\phi_\pi > 1$）の意味（→ 第6講で扱う想定）
- コストプッシュ・ショック下での「最適金融政策」がトレードオフをどう解くか（→ 第6講）
- 自然利子率推計の具体的方法（ウィリアムズ等のモデル）はブラックボックス
